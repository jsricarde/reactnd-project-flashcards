# UdaciCards: The flashcard application

**UdaciCards** is a mobile application(Android and iOS) build in React Native, that allows users to study collections of flashcards. The app will allow users to create different categories of flashcards called "decks", add flashcards to those decks, then take quizzes on those decks.

### Installation

**UdaciCards** requires [Node.js](https://nodejs.org/) v6+ to run, yarn in the last stable [version](https://yarnpkg.com/en/) and [expo(the cli for react-native)](https://docs.expo.io/versions/latest/introduction/installation/)

clone the git repo
```sh
$ git clone https://github.com/jsricarde/reactnd-project-flashcards.git
```

Install the dependencies and devDependencies and start the server.

```sh
$ cd reactnd-project-flashcards
$ npm install
```

if you're use yarn

```sh
$ cd reactnd-project-flashcards
$ yarn install
```

### Run

```sh
$ yarn start
```
 and follow the cli options
